INTRO TO PROGRAMMING

1. World Wide Web

The World Wide Web is a collection of HTML documents. HTML stands for Hyper Text Markup Language.
HTML documents are linked to each other by hyper-links.

2. Major Pieces of the Web

User <--> Internet <--> Servers

User - the user hosts the browser which is a program that displays the requested HTML documents found on the Internet.
Internet - world's largest network
Servers - computers that are optimized for storing files that make up the web
HTTP - main protocol of the web that is used to communicate across the web

3. HTML

An HTML document is made up of the following documents:

i. Text Content - is what you see
ii. Markup - is what the content looks like
iii. Reference to other documents
iv. links to other documents

3A. HTML Markup

An HTML is made up of elements which in turn consist tags. The term tag or element can be used interchangeably.The general structure of an element is as follows:

     <"Name">   "Contents"    <"/Name">
   Opening Tag                    Closing Tag

For example a bold element or tag can be written as:   ‹b› "hello world" ‹/b› will display as hello world

3B. HTML Attributes

An attribute is a property of a tag. For example:

The anchor tag ‹a› →    < a  href  = "url" >   "content"  < /a>

The image tag ‹img› → < img   src = "url" alt = "text" >

Here href and src are attributes of the respective anchor and image tags. The href attribute provides the link to the The image tag is an example of a void tag in that it does not need a closing tag.
The alt attribute in the image tag is used to display the text in case of a broken request where the requested image is not found.

3C. White Space

White space can be added using either the break tag -- ‹b› or the paragraph tag -- ‹p›.

3D. Inline Tags and Block Tags

Inline tags are tags which can can be written on the same line as the text. For instance the break tag is an example of an inline tag which outputs a line break where it is entered in the code.
The block tags like for instance the paragraph tag creates an invisible box around the content it contains.

3E. Container Elements

Container elements do nothing except contain content. Examples include ‹span› which is an inline element and ‹div› which is a block element.
